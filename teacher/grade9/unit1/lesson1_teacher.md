# Teacher Guide - Lesson 1

Duration: 80 min

## Lesson Flow
1. Think (10)
2. Mini Lecture (20)
3. Pair Activity (15)
4. Practice (20)
5. Reflection (15)

## Misconceptions
- ICT only means computers.
- Data and information are identical.

## Exit Ticket
Explain the difference between data and information in one sentence.
