# Lesson 2: Data, Information and Knowledge

## Big Question
How does raw data become useful information?

## Learning Goals
- Define data and information.
- Explain the data processing cycle.
- Distinguish data, information and knowledge.

## Warm-up
Count the number of students wearing sports shoes. Is your list data or information?

## Learn

### Data
Raw facts without context.

Examples:
- 21
- Blue
- 36.5°C

### Information
Processed data with meaning.

Example:
'The average classroom temperature is 27°C.'

### Knowledge
Understanding gained from information.

## Data Processing Cycle

Input → Processing → Output → Storage

## Activity
Measure classmates' heights and calculate:
- Tallest
- Shortest
- Average

Discuss what new information you discovered.

## AI Lab
Ask an AI:
'Explain the difference between data and information using school examples.'
Evaluate and improve the answer.

## Reflection
Why is accurate data important?

## Summary
Data becomes information after processing. Information helps people make decisions.
