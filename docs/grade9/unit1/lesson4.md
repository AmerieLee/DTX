# Lesson 4: Hardware and Software

## Big Question
How do hardware and software work together in a computer system?

## Learning Goals
- Distinguish hardware and software.
- Identify common hardware components.
- Explain how software controls hardware.

---

## Warm-up
List digital devices you used today. Identify hardware parts.

---

## Learn

### Hardware
Physical parts of a computer.

Examples:
- CPU
- Keyboard
- Mouse
- Monitor

### Software
Programs that run on hardware.

Examples:
- Operating system
- Apps
- Games

---

## Hardware vs Software

| Hardware | Software |
|----------|----------|
| Physical | Non-physical |
| Touchable | Code |

---

## AI Insight
Software tells hardware what to do.

Without software, hardware cannot function.

---

Choose a smartphone and list hardware + software components.
