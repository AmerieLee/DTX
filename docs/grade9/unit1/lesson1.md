# Lesson 1: What is Digital Technology?

## Essential Question
How does digital technology shape our daily lives?

## Learning Goals
- Define digital technology.
- Identify digital technologies around us.
- Explain the difference between data and information.

## 🤔 Think
List five digital technologies you used before arriving at school.

## 📖 Learn
Digital technology uses binary data to process, store and communicate information.
